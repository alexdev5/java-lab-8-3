# Лабораторна робота №8 (Варіант 3)
## Мета: 
Розробити консольний Java-додаток для рекурсивної обробки директорій з використанням багатопоточності, який здійснює пошук текстових файлів, підрахунок слів у файлах за заданою умовою, запис результатів у файл та виведення отриманої інформації в консоль.

## Опис
Напишіть консольний додаток Java, який рекурсивно обробляє вказану директорію, запускаючи для кожної піддиректорії окремий потік, з використанням високорівневого інтерфейсу для доступу до ресурсів:
- знайти в заданій директорії текстові файли, і підрахувати в кожному файлі кількість слів, що починаються із заданої літери. В деякий файл записати назви таких файлів і кількість знайдених слів. Вивести в консоль вміст створеного файлу.

## Виконання ```Main.java```
